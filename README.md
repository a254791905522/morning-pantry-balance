# Morning Pantry Balance Release Package

App name: Morning Pantry Balance

Bundle ID: com.rosewood.morningpantrybalance.game

Device support: iPhone only

Gameplay fact card:
- Core action: Tap food cards to build a breakfast plate that matches each day's calorie and macro targets.
- Main puzzle: Plan seven breakfasts from a shared pantry that must last the whole week.
- Macro balance: Carbs, protein, and fat each have their own tolerance; all three must match.
- Daily events: Gym workouts, hikes, and holidays shift the calorie target from level 4 onward.
- Progression: 50 levels (weeks), each with 7 days. Tutorial prompts guide levels 1-3.
- Monetization and network: offline, no ads, no in-app purchases, no account, no analytics, no tracking.
